# My Mail Server Config

Заполни этот файл своими значениями **до начала работы**.
AI будет читать его вместо того чтобы спрашивать тебя каждый раз.

> **Для AI-агента:** перед выполнением любой задачи прочитай этот файл.
> Незаполненные значения — это строки где значение совпадает с примером (`example.com`, `1.2.3.4`, `admin@example.com`, `mail2025`, пустые ячейки).
> Если нужное значение не заполнено — спроси пользователя, не угадывай и не используй примеры как реальные данные.
